# conflict-serializability


#### Instructions

* Code is written in Java on Eclipse
* JavaSE 1.7 is required
* git clone git@github.com:AhmetTanakol/conflict-serializability.git
* After cloning the repository, please import project in Eclipse
* To test a history, please change the variable called `history` in the program
* Click on `Run`
* Result will be displayed on console of Eclipse
* If history is serializable, it will print true otherwise false
